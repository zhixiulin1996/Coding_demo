This program plays hangman game. Users see a dashed word, trying to correctly figure the un-dashed word outby inputting one character each round.
If the user input is correct, show the updated word on console. Players have N_TURNS chances to try and win this game.

The list of words: 
"NOTORIOUS"
"GLAMOROUS"
"CAUTIOUS"
"DEMOCRACY"
"BOYCOTT"
"ENTHUSIASTIC"
"HOSPITALITY"
"BUNDLE"
"REFUND"
