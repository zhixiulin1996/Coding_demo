Boggle is a well-known board game in the USA! 
The game starts with a 4x4 grid of letter tiles, and players must connect adjacent letters on the board to form valid English words
(which is listed in the dictionary.txt file) 
In this program, I use the concept of backtracking to find all the answers.

Also, when reading the content of dictionary file, I use the trie data structure to increase the searching speed
